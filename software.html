Compressed Histogram of Gradients

 

For many mobile visual search applications, a query photo is taken by a mobile device and compared against a database on a remote server. The size of the data sent over the network needs to be as small as possible to reduce latency and improve user experience. We have studied descriptor compression techniques and shown that compressed descriptors can reduce query latency significantly in mobile image retrieval systems. Here, we make publicly available a Linux binary of a new low bitrate descriptor called Compressed Histogram of Gradients (CHoG), with the feature-location coding scheme cited below.
